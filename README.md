### Install

You don't need to install this project.

### Usage

To run the files, you can create a database (or open an existing one), select it, right click and select "Query tool"
After that, you can copy the content of the file you would like to execute in the blank page, and then press F5 to execute it.
For the schema, it will create a table
For the data, it will insert data to the table
For the query, it will show data based on the filters shown in the code.
